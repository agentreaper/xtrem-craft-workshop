# Clean code VS Dirty code

## Qu'est-ce que du code sale ?

- code non structuré 
- code dupliqué
- code spaghetti
- des libraries qui ne sont plus maintenues
- des commits qui ne veulent rien dire

## Qu'est-ce que du code propre ?

- Lisible 
- Maintenable 
- Facile à lire
- On plusieurs fichiers ayant des fonctions distinctes
- Sans surprise
